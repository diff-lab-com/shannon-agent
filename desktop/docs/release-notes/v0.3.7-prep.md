# Shannon Desktop v0.3.7 — UI Design Overhaul

This release ships a sprint-long audit-driven UI overhaul: 64 commits

### P0 — Honesty pass
- Removed misleading buttons, added dirty guards (Memory, OPC)

- New LoadingState, ErrorState, ConfirmDialog, SkeletonLoader primitives
- Focus-visible rings on icon-only buttons, chat/task actions, form fields
- Focus trap on 18 modal dialogs
- window.confirm → shared ConfirmDialog migration
- Virtualized chat message list (@tanstack/react-virtual)
- mod+n creates new chat session
- Editor Ask AI + file picker Browse + Edit mode + Save
- Anthropic + Ollama provider presets

### P2 — Page redesigns
- Chat / Memory / Extensions / Settings redesigned
- Chart palette uses --chart-series-* tokens

### P3 — Polish
- Brand icons + micro-interactions
- i18n audit (placeholders, severity labels, cancel aria, etc.)

- Tailwind 4 parser error (stray `)` in @custom-variant)
