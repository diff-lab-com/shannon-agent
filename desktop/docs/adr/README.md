## Format

Each ADR records:

- **Title**, **Date**, **Status**, **Supersedes**, **Superseded by**
- **Context** — what problem triggered the decision
- **Decision** — what was decided
- **Alternatives considered**
- **Consequences**
- **Follow-ups**

ADR status moves: Proposed → Accepted → Deprecated / Superseded. Once
Accepted, amend by writing a new ADR that supersedes it; do not edit the
historical record.
