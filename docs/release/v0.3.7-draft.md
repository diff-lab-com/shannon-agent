# v0.3.7 Release Notes (DRAFT)

**Status:** DRAFT — paste into `CHANGELOG.md` once PRs #50–#55 have all merged to `dev`. Tag `v0.3.7` after the dev → main PR merges.

**Scope:** Consumer-positioning pass — UI honesty fixes, shared primitives, design-system migration, and consumer-grade features (Plan Mode, Diff Preview, Voice Mode, Artifact Panel, Self-Improvement loop).

## Highlights (top of release blog)

- **UI honesty sweep.** Removed dead search, fixed Header bell wiring, gated the OPC human-in-the-loop approval correctly, killed misleading badges on Data Sources. The shell now shows what's actually backed by data.
- **Shared design primitives.** Modal, Card, Badge, Tooltip, DropdownMenu, Textarea added with MD3 tokens + dark mode. 30+ focus-ring fixes and 248 icon-name substitutions make the app feel consistent.
- **Plan Mode (Cmd+Shift+P).** Dedicated toggle in the chat composer — Shannon proposes a plan before touching files, with an exit-plan banner above the message list.
- **Diff Preview with syntax highlighting.** Files-changed summary bar, expand/collapse per file, "Review all" bulk action, highlight.js-powered per-line diff coloring with multi-line comment support.
- **Voice Mode (Phase 1-2).** MicButton + animated VoiceOrb in the chat composer. SpeechRecognition drives STT; the new TTS module wraps speechSynthesis with auto-cancel-on-unmount and voice matching. Provider abstraction (stub/webspeech/remote) ready for Whisper endpoints — HTTPS enforced.
- **Artifact Panel.** Renders HTML, SVG, Mermaid diagrams, and Document views with resize, fullscreen, and Ctrl+Shift+A cycle.
- **Self-Improvement loop (D6 Phases 1-5).** Shannon detects recurring tool-call patterns across past sessions, proposes them as skills, and an LLM-refinement step polishes the procedure before the user approves. Live `skill-catalog-changed` events refresh the Skills tab without a reload. Privacy opt-out toggle in Settings → Advanced.
