# Features

Shannon Code provides a rich set of features organized into these categories:

- **[Tool System](tools.md)** — File operations, bash execution, search, image analysis
- **[MCP Extensions](mcp.md)** — Model Context Protocol for extensibility
- **[Multi-Agent Orchestration](agents.md)** — Coordinated multi-agent workflows
- **[Permissions](permissions.md)** — Configurable approval modes
- **[Hooks](hooks.md)** — Event-driven customization
- **[Memory](memory.md)** — Persistent conversation memory
- **[Prompt Caching](caching.md)** — Three-layer Anthropic cache optimization
- **[Internationalization](i18n.md)** — 10-language UI support
