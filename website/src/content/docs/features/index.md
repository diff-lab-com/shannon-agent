# Features

Shannon Code provides a rich set of features organized into these categories:

- **[Tool System](tools/)** — File operations, bash execution, search, image analysis
- **[MCP Extensions](mcp/)** — Model Context Protocol for extensibility
- **[Multi-Agent Orchestration](agents/)** — Coordinated multi-agent workflows
- **[Permissions](permissions/)** — Configurable approval modes
- **[Hooks](hooks/)** — Event-driven customization
- **[Memory](memory/)** — Persistent conversation memory
- **[Prompt Caching](caching/)** — Three-layer Anthropic cache optimization
- **[Internationalization](i18n/)** — 10-language UI support
