# Change Log

## [0.1.0] — 2026-06-02

### Added

- Chat WebView panel with Markdown rendering and syntax highlighting
- NDJSON subprocess communication with Shannon CLI
- Multi-provider support: Anthropic, OpenAI, Ollama, DeepSeek
- File diff viewer with accept/reject workflow
- Status bar indicator (disconnected, connected, working, error)
- Keyboard shortcuts: `Ctrl+Shift+S` (open chat), `Ctrl+Shift+Enter` (send), `Escape` (stop)
- Extension configuration: CLI path, provider, model, API key
- First-activation welcome message
